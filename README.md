# 🎮 2학년 1학기 개인 유니티 프로젝트

<br>

## 📌 프로젝트 소개
Unity를 활용하여 제작한 개인 게임 프로젝트입니다.  
플레이어는 토끼를 조작하여 숲 속을 이동하며 당근을 수집하고,  
곰을 피해 점수를 획득하는 게임입니다.

<br>

## 🎬 시연 영상
[![Demo](https://img.youtube.com/vi/BPRVbsQ0YaA/0.jpg)](https://youtu.be/BPRVbsQ0YaA)

<br>

## 🛠 사용 기술
- Unity  
- C#  

<br>

## ✨ 주요 기능
- 🐰 토끼 캐릭터 이동 및 조작  
- 🐻 플레이어를 추적하는 곰 AI  
- 🥕 랜덤 위치에 생성되는 당근 시스템  
- 🌲 숲 환경 및 오브젝트 배치  

<br>

## 🧑‍💻 구현 내용
- 플레이어 입력을 기반으로 토끼 이동 시스템 구현  
- 곰이 플레이어를 추적하도록 간단한 AI 로직 구현  
- 당근이 맵 내 랜덤 위치에 생성되도록 설정  
- 점수 시스템을 통해 수집 요소를 게임에 반영  
- 숲 환경을 구성하여 게임의 분위기 조성  

<br>

## 🎮 조작 방법
- W / A / S / D : 이동  
- Shift : 달리기  
- Space : 점프    

<br>

## 🎯 게임 목표
곰을 피해 최대한 많은 당근을 수집하여 높은 점수를 기록하는 것이 목표입니다.

<br>

## 📌 느낀 점
Unity를 활용하여 직접 게임을 제작하면서  
캐릭터 이동, AI 동작, 랜덤 생성 시스템 등  
게임의 기본적인 구성 요소를 구현해볼 수 있었습니다.  

특히 적이 플레이어를 추적하는 기능과  
아이템이 랜덤으로 생성되는 시스템을 구현하면서  
게임 로직 설계에 대한 이해를 높일 수 있었습니다.
